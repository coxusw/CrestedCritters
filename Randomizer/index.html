FAQ & Name Tools

How the official randomizer works, how usernames work, and how to collect giveaway entry names.

How the official randomizer works

The public wheel page does not pick the winner. Backend verifies the runner login, creates the official spin, saves it to Database, and creates a public result code.

Why this is more trustworthy

The result is determined and saved before it is displayed. The result page includes the original entry list, spin history, final winner, and verification code.

Runners and players alike can verify authenticity with the verify this result button.

Duplicate entries

Duplicate names are kept as separate entries. If someone earned three entries, list their name three times.

Name Collector JavaScript

This helper collects visible/loadable names from Facebook or Instagram pages you can access. It cannot run inside the native Facebook or Instagram app and it does not bypass privacy settings.

Do not paste random JavaScript from websites you do not trust. This helper is provided by Crested Critters for collecting visible giveaway names only.

How to Use the Name Collector Helper

The Name Collector Helper now opens a small menu when it runs. You can choose whether you are collecting names from a visible list such as reactions/followers, or whether you want to collect commenters from a post.

Important: Use a browser, not the app

This works best in Chrome, Safari, Edge, or Firefox. It usually will not work inside the Facebook or Instagram app.

Best Method: Bookmarklet

  1. Copy the Bookmarklet Code — Best for Most Users.
  2. Create a browser bookmark named Collect Giveaway Names.
  3. Paste the copied code into the bookmark URL/address field.
  4. Save the bookmark.
  5. Open Facebook or Instagram in your browser.
  6. Open the reactions list, followers list, likes list, or comments section you want to collect from.
  7. Click the Collect Giveaway Names bookmark.
  8. Choose the collection mode in the helper menu.
  9. Click Start Collecting.
  10. Review the list, then click Copy Names.
  11. Paste the names into the Randomizer Wheel.

Collecting Facebook post reactions

  1. Open Facebook in a browser.
  2. Open the giveaway post.
  3. Click the reactions/likes count so the list of people opens.
  4. Run the Name Collector Helper.
  5. Choose Reactions / followers / visible name list.
  6. Click Start Collecting.

Collecting Facebook post comments

  1. Open Facebook in a browser.
  2. Open the giveaway post.
  3. Open the comments section and load as many comments as you need.
  4. Run the Name Collector Helper.
  5. Choose Post commenters from loaded comments.
  6. Click Start Collecting.
  7. Review the results. Facebook layouts change often, so remove anything that is not a real entrant.

Collecting Instagram likes, followers, or comments

  1. Open Instagram in a browser.
  2. Open the likes list, followers list, or comments section.
  3. Run the Name Collector Helper.
  4. Choose the mode that matches what you are collecting.
  5. Review the names or usernames before generating the official wheel.

Alternative Method: Browser Console

The Console Code is mainly for advanced users. Most people should use the bookmarklet code.

Always review the names

The Name Collector Helper is only a helper. The giveaway runner is responsible for reviewing the final entry list before generating the official result.

  • Remove words that are not names.
  • Check that intentional duplicate entries are correct.
  • Check for missing people before generating the official result.
  • Remember: once the official result is generated, the saved result shows the exact entry list that was used.
